# Study Session Tracker

This is a simple command-line Python app to help track daily study sessions.

## Purpose

I built this tool while grinding data structures and algorithms to prepare for coding interviews and internship applications. I use it to keep a log of what I study each day, how long I studied, and what topics I focused on. It's part of my effort to stay consistent and build discipline as I work through LeetCode and core DSA concepts.

## Features

- Add a new study session (subject, minutes, notes)
- View a list of all past sessions
- See a summary of total study time in the last 7 days
- Saves data to a local `study_log.json` file

## Usage

```bash
python study_tracker.py
